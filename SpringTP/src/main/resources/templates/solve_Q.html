■ 나머지 구하기

정수 num1, num2가 매개변수로 주어질 때, num1을 num2로 나눈 나머지를 return 하도록 solution 함수를 완성해주세요.

제한사항

  • 0 < num1 ≤ 100
  • 0 < num2 ≤ 100

입출력 예

num1num2result
321
1050

입출력 예 설명

num1이 3, num2가 2이므로 3을 2로 나눈 나머지 1을 return 합니다.

num1이 10, num2가 5이므로 10을 5로 나눈 나머지 0을 return 합니다.
